          Example

          *

          element
          p
          Selects all <p> elements
        

        

          element,element,..
          div, p
          Selects all <div> elements and all <p> elements

All CSS Combinator Selectors

      

      

        

          Selector
          Example
          Example description
        

        

          element element
          div p
          Selects all <p> elements inside <div> elements
        

        

          element>element
          div > p
          Selects all <p> elements where the parent is a <div> element
        

        

          element+element
          div + p
          Selects the first <p> element that are placed immediately after <div> elements
        

        

          element1~element2
          p ~ ul
          Selects every <ul> element that are preceded by a <p> element

      Pseudo already defined.

      ## Pseudo-class
      A pseudo-class is used to define a special state of an element.
      ```css
      syntax:
      selector:pseudo-class {
        property: value;
      }
      
      example:
      a:link {
        color: #FF0000;
      }
      
      a:visited {
        color: #00FF00;
      }
      
      a:hover {
        color: #FF00FF;
      }
      
      a:active {
        color: #0000FF;
      }
      ```
      ## Pseudo-elements
      A CSS pseudo-element is used to style specified parts of an element.
      ```css
      syntax:
      selector::pseudo-element {
        property: value;
      }
      
      example:
      p::first-line {
        color: #ff0000;
        font-variant: small-caps;
      }
      ```

### for迴圈

      

        ```js
        for (var i=0; i<x.a.length; i++) {
                  r[i] = x.a[i]+y.a[i]
              }
        ```
      

      1. i 從 0 開始迭代
        
      2. 當 i 符合 i < x.a.length 才迭代
      
      3. 迭代完成後 i +1
      
      即：當 i = x.a.length 時，停止迭代

      

      
### method

      

        ```js
        class Vector {
            constructor(array) {
              this.a = array
            }
            add(y) {
              var r = []
              var x = this
              for (var i=0; i<x.a.length; i++) {
                  r[i] = x.a[i]+y.a[i]
              }
              return new Vector(r)
            }
          }
        ```
      

      add 方法 return 了新的 Vector 物件
        
      r = 新物件的 array = 建構了新物件的 a

      

      
### super

      

        ```js
        class Animal {
          constructor(name) {
            this.name = name
          }
          say() {}
        }
        
        class Dog extends Animal {
          constructor() {
            super('狗')
          }
          say() { console.log(this.name, '汪汪') }
        }
        ```
      

      子類別中，利用 super 呼叫父類別之建構函數，作為自己的建構函數。
        
      super 之值即為建構函數之引數。
